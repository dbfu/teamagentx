# 用户手册

这套文档按 TeamAgentX 当前界面和代码实现整理，每个功能都有独立页面。建议新用户按顺序阅读，老用户可以直接进入对应功能页。

## 推荐阅读顺序

1. [首次设置](/user-guide/first-run)：完成本地 AI 工具检测、默认模型配置和账号创建。
2. [模型管理](/user-guide/models)：准备文本、图片、语音等模型配置。
3. [助手管理](/user-guide/agents)：创建助手，配置 Agent、模型、思考模式、图片能力和提示词。
4. [群聊与消息](/user-guide/chatrooms)：创建群组，通过 `@助手名` 协作，查看任务执行过程。
5. [快速对话](/user-guide/quick-chat)：为一次性任务创建临时会话。
6. [技能管理](/user-guide/skills)：导入、查看并安装技能到助手。
7. [定时任务](/user-guide/cron-tasks)：让群聊按计划自动发送任务并触发助手。
8. [频道集成](/user-guide/integrations)：把外部平台机器人绑定到群聊。
9. [设置与多端连接](/user-guide/settings)：管理个人信息、外观、提示音、桌面端和移动端连接。

## 常用入口对照

| 功能 | 界面入口 | 适合做什么 |
| --- | --- | --- |
| 群聊协作 | 左侧 **消息** | 日常和多个助手一起处理任务 |
| 创建群组 | 左侧 **+** → **创建群组** | 建立一个长期协作空间 |
| 创建助手 | 左侧 **+** → **创建助手** 或 **助手** 页面 | 为不同角色配置 AI 成员 |
| 模型管理 | 左侧 **模型** | 管理 API URL、API Key、模型 ID 和默认模型 |
| 技能管理 | 左侧 **技能** | 管理共享技能库和助手技能安装 |
| 频道集成 | 左侧 **频道** | 接入 Telegram、飞书、钉钉、企业微信、QQ |
| 个人设置 | 左下角头像或左侧 **设置** | 修改用户信息、主题、通知和多端连接 |

## 概念关系

- **模型** 是助手调用 AI 服务的配置。
- **助手** 是执行任务的 AI 成员，可以绑定模型、Agent 工具、提示词和技能。
- **群聊** 是用户和多个助手协作的工作空间。
- **工作目录** 决定助手执行文件操作和命令时所在的位置。
- **技能** 是可复用的任务说明和上下文，可以安装到一个或多个助手。
- **定时任务** 会在指定时间向群聊发送消息，并按配置触发助手。
- **频道** 负责把外部平台消息同步到 TeamAgentX 群聊。
