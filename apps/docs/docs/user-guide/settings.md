# 设置与多端连接

设置页用于管理个人信息、外观、通知、桌面端工具运行方式，以及移动端和网页访问。

## 页面入口

点击左下角头像或进入左侧 **设置**。

## 个人信息

可以修改：

- **用户名**
- **头像**

页面会显示预览。点击 **保存修改** 后生效。

## 外观

外观设置包含两类：

### 明暗模式

- 浅色
- 深色
- 跟随系统

### 品牌色

可选择商务蓝、石墨灰、灵感紫、翡翠绿、曜石红等主题。切换后会影响按钮、导航高亮和强调色。

## 通知

可以开关 **消息提示音**。开启后，收到助手回复时播放提示音。

## 客户端设置

桌面端会显示 **客户端** 区域。

- **开机自动启动**：开启后登录系统时自动启动 TeamAgentX。
- 如果当前系统不支持客户端内设置，会显示提示。

## 终端设置

macOS 桌面端会显示 **终端** 区域。

这里决定在群工作目录菜单选择 **终端** 时使用哪个应用打开。

## AI 工具运行方式

桌面端会显示 **AI 工具运行方式**。

页面会检测 Claude、Codex 等工具的：

- CLI 状态和版本。
- SDK 状态和版本。
- 当前优先运行方式。

可以点击：

- **重新检测**
- **安装 SDK**
- **重新安装 SDK**

规则：

- 已安装应用本地 SDK 时，Claude / Codex 助手优先使用 SDK。
- 未安装 SDK 但宿主机 CLI 可用时，会继续使用宿主机 CLI。

## 移动端连接

设置页会在可用环境中显示 **移动端连接**。

操作步骤：

1. 确认手机和电脑在同一局域网。
2. 在 **局域网地址** 中选择正确 IP。
3. 必要时修改 **服务器地址**。
4. 点击 **生成二维码**。
5. 使用手机 TeamAgentX App 或扫码应用扫描二维码。

如果无法连接，检查：

- 手机和电脑是否在同一网络。
- 选择的 IP 是否是当前电脑的局域网 IP。
- 防火墙是否阻止访问。
- 后端服务是否还在运行。

## 网页访问

桌面端在具备本地服务地址、登录 token 和用户信息时，会显示 **网页访问**。

点击 **打开网页** 会用当前登录信息生成访问链接，并在外部浏览器打开。

## 客户端更新

桌面端会显示当前版本，并提供 **检查更新**。

如果有更新，左侧导航底部也可能显示更新提示。

## 官网与文档

设置页底部包含：

- **官网主页**
- **使用文档**

这些入口会使用外部浏览器打开。
