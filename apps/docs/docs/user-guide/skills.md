# 技能管理

技能是可复用的任务说明和上下文。安装到助手后，助手可以在执行任务时参考技能中的规则、流程和文件。

## 页面入口

左侧点击 **技能** 进入技能管理页。

页面顶部包含：

- **导入技能**
- **刷新**
- **搜索技能**
- **打开目录**（桌面端）
- **复制技能库路径**

如果当前没有技能，页面会提示在群聊中 `@技能管理` 创建技能。

## 共享技能列表

技能列表会显示：

- 技能名称。
- 技能描述。
- 技能来源：用户创建、外部导入或外部。
- 已安装到多少个助手。
- **查看**
- **安装**

展开已安装的技能，可以看到安装到哪些助手，并可从某个助手移除。

## 查看技能内容

点击 **查看** 后，会打开技能内容窗口。

窗口包含：

- 技能名称和描述。
- 文件列表。
- 文件内容预览。
- 来源和文件数量。

安装前建议先查看 `SKILL.md`，确认技能行为符合当前团队规范。

## 安装技能到助手

点击技能右侧 **安装**，选择一个或多个目标助手。确认后技能会安装到这些助手。

助手详情页的 **Skills** 标签也可以安装新 Skill 或卸载已有 Skill。

## 移除技能

有两种方式：

- 在技能页展开技能，点击已安装助手旁的移除按钮。
- 在助手详情页 **Skills** 标签中点击删除按钮。

移除后，该助手后续任务不再使用该技能。

## 导入外部技能

点击 **导入技能** 打开导入窗口。窗口会按外部工具分组展示可导入技能，并提供搜索和全选。

### 扫描外部技能

系统会列出外部技能目录中的技能。

- 已导入的技能会标记 **已导入**，不可重复选择。
- 未导入的技能可以勾选后批量导入。

### 本地文件夹

切换到 **本地文件夹** 标签，可以选择一个本地技能文件夹。文件夹根目录必须包含 `SKILL.md`。

### 导入方式

- **复制导入**：完整复制技能文件到本地共享技能库，之后独立管理。
- **软连接导入**：创建符号链接指向外部技能目录，外部目录更新后会同步变化。Windows 下不显示软连接入口。

## 创建技能

当群聊中存在技能管理助手时，可以发送：

```text
@技能管理 请根据我们刚才整理的发布流程创建一个技能。
```

创建完成后，技能会进入共享技能库，再由用户安装到需要的助手。

## 使用建议

- 把稳定、重复的流程做成技能。
- 技能名称要能表达用途，例如 `release-checklist`、`frontend-review`。
- 一个技能只解决一类问题，不要把无关规则混在一起。
- 技能更新后，确认已安装助手是否需要重新读取或重新安装。
