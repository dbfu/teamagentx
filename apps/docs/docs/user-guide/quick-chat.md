# 快速对话

快速对话用于和单个助手创建临时会话。它不需要先创建正式群组，适合一次性任务、快速实验和临时文件处理。

## 发起快速对话

可以从两个入口发起：

- **助手** 页面中的快速对话按钮。
- 助手详情页顶部的 **快速对话**。

弹框会显示目标助手，并要求确认工作目录。

## 工作目录

快速对话弹框包含 **工作目录** 输入框。

- 留空：使用默认目录策略，每次快速对话创建独立会话目录。
- 填写路径：该路径作为本次快速对话工作目录。
- 桌面端可以点击目录按钮选择本地文件夹。

如果助手本身配置了默认目录，弹框会显示 **助手默认目录** 作为参考。

## 快速对话的行为

快速对话创建后会出现在消息列表中，但它和普通群聊不同：

- 消息会直接发送给创建时选择的助手。
- 不需要手动 `@`。
- 不允许添加其他助手。
- 群设置中会提示这是快速对话群聊。

## 会话记录

进入助手详情页的 **会话记录** 标签，可以查看该助手创建过的快速对话。

每条记录会显示：

- 会话名称。
- 状态：活跃或已归档。
- 创建时间。
- **打开** 按钮。

## 适合场景

- 临时让助手查看一个目录。
- 快速生成脚本、命令或文案。
- 测试一个新助手是否可用。
- 排查一个不需要长期保留上下文的问题。

## 不适合场景

以下情况建议创建普通群聊：

- 需要多个助手协作。
- 需要长期保留固定群规则。
- 需要绑定外部平台机器人。
- 需要在同一工作空间内反复执行定时任务。
