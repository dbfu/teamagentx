# 频道集成

频道集成用于把外部平台机器人连接到 TeamAgentX 群聊。外部平台收到的消息会进入绑定群聊，群聊中的助手也可以参与外部会话。

## 页面入口

左侧点击 **频道**，进入 **外部平台频道** 页面。

当前页面支持的平台包括：

- Telegram
- 飞书
- 钉钉
- 企业微信
- QQ

平台列表来自后端配置，实际显示以页面为准。

## 服务公网地址

页面顶部是 **服务公网地址**。

企业微信、QQ、Telegram 等 Webhook 场景通常需要一个外部平台可以访问的公网地址。

填写示例：

```text
https://your-domain.com
```

如果未配置，页面会提示 Webhook 场景需要该地址。

## 创建机器人实例

1. 选择目标平台。
2. 点击机器人列表中的新建入口。
3. 查看页面内的 **接入说明**。
4. 填写 **机器人实例名称**。
5. 填写平台要求的凭证字段，例如 Token、Secret、App ID。
6. 选择 **创建后直接绑定群聊**，或选择 **暂不绑定**。
7. 点击 **创建机器人实例**。

编辑机器人时，密钥字段不会回显；留空表示保持原值。

## 绑定群聊

一个机器人实例最多绑定一个群聊；一个群聊可以同时绑定多个平台机器人。

绑定后：

- 外部平台任意会话消息会进入绑定群聊。
- 群聊连接概览会显示已连接机器人数量。
- 群设置中的 **外部平台机器人绑定** 也会显示当前群聊已绑定机器人。

如果机器人已经绑定其他群聊，再绑定到新群聊时会出现换绑确认。

## Webhook

部分平台需要把 Webhook 地址复制到平台后台。频道页面支持复制机器人 Webhook。

使用前确认：

- 服务公网地址已配置。
- 外部平台能访问该地址。
- 平台后台的回调地址与页面生成地址一致。
- 机器人凭证正确。

## 最近同步事件

右侧 **最近同步事件** 展示最近的流入或流出消息。

每条事件包含：

- 平台。
- 方向：流入或流出。
- 消息预览。
- 时间。
- 助手名称。
- 外部会话 ID。
- 状态：成功或失败。
- 错误信息。

排查外部平台问题时，先查看这里是否有事件记录。

## 飞书快捷入口

飞书平台接入说明中提供快捷创建机器人入口。需要跳转到飞书开放平台完成相关配置。

## 使用建议

- 先在一个测试群聊中绑定机器人，确认消息双向同步正常。
- 生产环境使用稳定公网域名，不要使用临时地址。
- 密钥字段只在需要变更时重新填写。
- 一个群聊可绑定多个平台，但建议用清晰的机器人实例名称区分来源。
