# 配置指南

这里汇总 TeamAgentX 的常用配置说明。普通用户通常只需要关注模型配置；管理员或部署人员再查看环境变量和智能体设置细节。

## 配置入口

- [LLM 配置](/configuration/llm)：创建和管理助手使用的模型配置。
- [智能体设置](/configuration/agent-settings)：了解提示词、工作目录、助手类型和房间记忆。
- [环境变量](/configuration/env)：查看服务端、桌面端和前端相关环境配置。

## 推荐阅读顺序

1. 先阅读 [用户使用手册](/user-guide/)，完成首次配置和基本使用。
2. 需要新增模型时，阅读 [LLM 配置](/configuration/llm)。
3. 需要调整助手行为时，阅读 [智能体设置](/configuration/agent-settings)。
4. 需要部署或排查服务配置时，阅读 [环境变量](/configuration/env)。
