# 智能体设置

## 概述

智能体设置定义智能体的行为和能力，包括提示词、工具、工作目录等。

## 提示词配置

提示词定义智能体的角色和行为：

### 基本结构

```
你是一个[角色描述]，帮助用户[功能描述]。
你可以执行[能力描述]。
遇到问题时，请[处理方式]。
```

### 示例

代码助手提示词：

```
你是一个代码助手，帮助用户编写和调试代码。
你可以执行 Shell 命令来检查文件、运行测试。
遇到问题时，请分析错误信息并提供解决方案。
保持代码整洁、遵循最佳实践。
```

### 最佳实践

- 明确角色和能力
- 描述具体行为
- 提供错误处理指导
- 设置边界和限制

## 工作目录

工作目录决定智能体执行操作的文件系统范围。

### 配置层级

1. **智能体工作目录**: 智能体的默认工作目录
2. **聊天室工作目录**: 群组共享目录
3. **快速对话工作目录**: 临时目录

优先级：快速对话 > 聊天室 > 智能体默认

### 设置建议

- 使用绝对路径
- 确保目录有足够权限
- 避免使用系统敏感目录

## 智能体类型

### 内置智能体

使用 LangChain 执行器，适合：

- 对话型任务
- 工具调用
- 简单自动化

### ACP 智能体

使用 Claude 或 Codex 工具，适合：

- 复杂代码任务
- 多步骤操作
- 高级推理

## 工具配置

内置智能体可以配置工具：

- Shell 命令执行
- 文件操作
- API 调用

ACP 智能体的工具由 ACP 工具自动管理。

## 分类管理

将智能体按功能分类：

- 创建分类
- 设置分类名称和图标
- 调整分类排序
- 将智能体分配到分类

## 房间记忆

智能体在聊天室中维护记忆：

- 自动总结历史对话
- 保留重要信息
- 压缩和优化记忆

记忆参数通过环境变量配置。
