# 功能介绍

TeamAgentX 的用户功能已经按页面拆成完整使用手册。这里作为功能入口目录，方便快速跳转。

## 用户功能

- [首次设置](/user-guide/first-run)：检测 Claude / Codex，本地或手动配置默认模型，创建账号。
- [模型管理](/user-guide/models)：管理文本、图片、语音模型，测试连接，导入导出 JSON。
- [助手管理](/user-guide/agents)：创建助手，配置 Agent、模型、提示词、思考模式、图片能力和 Skills。
- [群聊与消息](/user-guide/chatrooms)：创建群组，`@` 助手，设置触发模式、群规则、工作目录和任务看板。
- [快速对话](/user-guide/quick-chat)：为单个助手创建临时会话，处理一次性任务。
- [技能管理](/user-guide/skills)：导入技能、查看文件内容、安装到助手或移除。
- [定时任务](/user-guide/cron-tasks)：配置 Cron、固定间隔或一次性任务，并查看执行历史。
- [频道集成](/user-guide/integrations)：接入 Telegram、飞书、钉钉、企业微信、QQ 等外部机器人。
- [设置与多端连接](/user-guide/settings)：个人信息、外观、提示音、桌面端工具、移动端二维码和网页访问。
