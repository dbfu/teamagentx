# 功能介绍

TeamAgentX 是一个多智能体协作平台，提供类似 Feishu 的聊天界面，让用户可以与多个 AI 智能体进行交互和协作。

## 核心功能

### 智能体管理

系统内置多种智能体，也支持用户自定义创建智能体：

- **内置智能体**: 预配置的助手，如通用助手、代码助手等
- **ACP 智能体**: 使用 Claude、Codex 等 ACP 工具的高级智能体
- **自定义智能体**: 用户可以创建自己的智能体，配置提示词、工具、工作目录等

详细说明请参考 [智能体管理](/features/agents)。

### 聊天室

聊天室是智能体协作的核心场所：

- 创建和管理聊天室
- 添加多个智能体到聊天室
- 通过 `@agent-name` 提及智能体
- 消息回复和引用
- 执行记录追踪
- 未读消息提醒

详细说明请参考 [聊天室](/features/chatrooms)。

### 快速对话

快速对话模式适合临时性的任务：

- 无需创建聊天室
- 自定义工作目录
- 快速实验和测试
- 自动清理会话

详细说明请参考 [快速对话](/features/quick-chat)。

### 定时任务

在聊天室中创建定时执行的任务：

- 支持 CRON 表达式
- 自动执行和记录
- 任务状态追踪
- 执行历史查看

详细说明请参考 [定时任务](/features/cron-tasks)。

## 智能体能力

智能体可以执行多种操作：

- **Shell 命令**: 执行系统命令、文件操作等
- **ACP 工具**: 使用 Claude、Codex 等高级工具
- **知识检索**: 访问聊天室记忆和历史消息
- **任务移交**: 智能体之间可以移交任务

## 多平台支持

- **Web**: 基于 React 的前端应用
- **桌面版**: Electron 封装，支持 macOS/Windows
- **移动端**: Flutter 应用，支持 iOS/Android
