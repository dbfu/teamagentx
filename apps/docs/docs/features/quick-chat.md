# 快速对话

## 概述

快速对话是一种临时性的对话模式，适合快速实验和一次性任务。

## 特点

- **无需创建聊天室**: 直接开始对话
- **自定义工作目录**: 可以指定临时工作目录
- **自动清理**: 会话结束后可以选择清理

## 使用方法

### Web 端

1. 点击导航栏的「快速对话」按钮
2. 选择要使用的智能体
3. 可选择设置临时工作目录
4. 开始对话

### 桌面版

桌面版支持快速对话，可以：
- 选择本地目录作为工作目录
- 快速执行命令和任务

### 移动端

移动端同样支持快速对话，方便随时使用。

## 工作目录

快速对话可以设置临时工作目录：

- 选择现有目录
- 创建新目录
- 目录仅在会话期间有效

## 会话管理

### 会话列表

可以在快速对话列表中查看所有会话。

### 会话清理

完成后可以：
- 保留会话记录
- 清理会话目录
- 删除整个会话

## 适用场景

快速对话适合：

- **快速实验**: 测试智能体能力
- **临时任务**: 一次性文件操作
- **代码测试**: 快速运行代码
- **问题排查**: 快速诊断问题
