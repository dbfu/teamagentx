# 智能体管理

## 概述

智能体（Agent）是 TeamAgentX 的核心组件，负责与用户交互、执行任务、协调工作。

## 智能体类型

### 内置智能体 (Builtin)

内置智能体使用 LangChain 执行器，支持：

- 自定义提示词
- 工具调用
- 对话历史管理
- 房间记忆

### ACP 智能体

ACP 智能体使用外部 ACP 工具，目前支持：

- **Claude**: 使用 Anthropic Claude API
- **Codex**: 使用 OpenAI Codex SDK

ACP 智能体具有更强大的能力，可以执行复杂的代码任务。

## 创建智能体

1. 点击左侧导航栏的「助手管理」
2. 点击「创建助手」按钮
3. 配置智能体信息：

   - **名称**: 智能体显示名称
   - **描述**: 智能体功能描述
   - **类型**: 选择内置或 ACP
   - **ACP 工具**: 如果是 ACP 类型，选择 Claude 或 Codex
   - **提示词**: 智能体的系统提示
   - **工作目录**: 智能体执行任务的基础目录
   - **LLM 配置**: 选择语言模型配置

4. 保存智能体

## 智能体配置

### 提示词

提示词定义智能体的行为和能力。例如：

```
你是一个代码助手，帮助用户编写和调试代码。
你可以执行 Shell 命令来检查文件、运行测试。
遇到问题时，请分析并提供解决方案。
```

### 工作目录

工作目录是智能体执行操作的基础路径：

- 内置智能体可以配置固定工作目录
- 聊天室可以设置群组工作目录
- 快速对话支持临时工作目录

### LLM 配置

LLM 配置定义智能体使用的语言模型：

- 支持自定义 API endpoint
- 支持 Anthropic 或 OpenAI 协议
- 配置 API 密钥和模型名称

## 分类管理

智能体可以按分类组织：

- 创建分类
- 设置分类排序
- 将智能体分配到分类

## 使用智能体

在聊天室中使用智能体：

1. 将智能体添加到聊天室
2. 在消息中使用 `@智能体名称` 提及
3. 智能体会自动响应并执行任务
