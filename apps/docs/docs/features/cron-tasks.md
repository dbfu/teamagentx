# 定时任务

## 概述

定时任务允许在聊天室中创建定期执行的任务，支持 CRON 表达式配置。

## 创建定时任务

1. 进入聊天室
2. 点击「定时任务」按钮
3. 配置任务：

   - **任务名称**: 标识任务
   - **触发表达式**: CRON 表达式
   - **任务内容**: 要执行的内容
   - **智能体**: 执行任务的智能体

4. 保存任务

## CRON 表达式

支持标准 CRON 表达式：

```
* * * * *
│ │ │ │ │
│ │ │ │ └─ 星期几 (0-6, 0=周日)
│ │ │ └─── 月份 (1-12)
│ │ └───── 日期 (1-31)
│ └─────── 小时 (0-23)
└───────── 分钟 (0-59)
```

示例：
- `0 9 * * *`: 每天 9:00 执行
- `30 14 * * 1-5`: 工作日 14:30 执行
- `0 0 1 * *`: 每月 1 日 0:00 执行

## 任务管理

### 查看任务列表

在聊天室中查看所有定时任务：

- 任务名称和状态
- 下次执行时间
- 执行历史

### 启用/禁用任务

可以临时禁用或启用任务。

### 编辑任务

修改任务配置、执行时间等。

### 删除任务

删除不再需要的任务。

## 执行记录

每次任务执行都会记录：

- 执行时间
- 执行结果
- 错误信息（如有）
- Token 使用

可以在执行历史中查看详细信息。

## 使用场景

定时任务适合：

- **定期报告**: 每日生成数据报告
- **系统监控**: 定期检查系统状态
- **数据备份**: 定期执行备份操作
- **提醒通知**: 定期发送提醒消息
