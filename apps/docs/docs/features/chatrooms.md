# 聊天室

## 概述

聊天室是 TeamAgentX 的主要交互场所，用户可以在这里与智能体进行对话和协作。

## 创建聊天室

1. 点击左侧导航栏的「创建聊天室」
2. 输入聊天室名称
3. 选择要添加的智能体
4. 配置聊天室设置：
   - **群组工作目录**: 智能体执行任务的公共目录
   - **默认智能体**: 输入消息时默认提及的智能体
   - **触发模式**: 智能体响应触发方式

## 聊天室功能

### 提及智能体

在消息中使用 `@智能体名称` 提及智能体：

```
@代码助手 请帮我检查这个函数的实现
```

被提及的智能体会自动响应并执行任务。

### 多智能体协作

可以同时提及多个智能体：

```
@助手A @助手B 请协作完成这个任务
```

智能体之间可以：
- 共享对话上下文
- 移交任务
- 协调工作

### 消息回复

支持回复特定消息：

1. 点击消息右侧的回复按钮
2. 输入回复内容
3. 回复会关联到原消息

### 执行记录

智能体执行的任务会记录：

- 执行状态（进行中/完成/失败）
- 事件序列
- Token 使用统计
- 执行时长

点击消息可以查看详细的执行记录。

### 未读消息

系统自动追踪未读消息：

- 显示未读消息数量
- 点击聊天室自动标记已读
- 支持手动标记已读

## 聊天室设置

### 群组工作目录

设置智能体执行任务的公共目录，所有智能体在此目录下工作。

### 默认智能体

设置默认提及的智能体，发送消息时自动添加提及。

### 触发模式

- **手动触发**: 只有明确提及才响应
- **自动触发**: 所有消息都触发响应

## 聊天室管理

### 固定聊天室

将常用聊天室固定到顶部，方便快速访问。

### 离开聊天室

可以从聊天室离开，不再接收消息通知。

### 删除聊天室

删除聊天室会同时删除所有消息和记录。请谨慎操作。
